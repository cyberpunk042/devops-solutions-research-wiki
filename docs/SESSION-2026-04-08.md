## Entry Points

### Pipeline (primary вҖ” use `.venv/bin/python`)
| Command | Purpose |
|---------|---------|
| `pipeline post` | Post-ingestion chain (index вҶ’ manifest вҶ’ validate вҶ’ obsidian вҶ’ lint) |
| `pipeline fetch URL [URL...]` | Fetch URLs into raw/ |
| `pipeline run URL [URL...]` | Parallel fetch + post-chain |
| `pipeline gaps` | Gap analysis (orphans, thin pages, weak domains, open questions) |
| `pipeline crossref` | Cross-reference analysis (backlinks, bridges, comparisons) |
| `pipeline scaffold <type> <title>` | Create page from template |
| `pipeline evolve --score` | Rank evolution candidates (deterministic) |
| `pipeline evolve --scaffold --top N` | Scaffold top N candidates |
| `pipeline evolve --auto --backend openai` | Generate via local model |
| `pipeline evolve --auto --backend claude-code` | Write prompt queue |
| `pipeline evolve --execute` | List/process prompt queue |
| `pipeline evolve --review` | List seed pages ready for promotion |
| `pipeline evolve --dry-run --top 1` | Preview generation prompt |
| `pipeline chain health` | Post вҶ’ gaps вҶ’ crossref |
| `pipeline chain evolve` | Score вҶ’ scaffold вҶ’ post |
| `pipeline chain evolve-auto` | Score вҶ’ scaffold вҶ’ generate (local) вҶ’ post |
