# User Directive — 2026-04-08 — No New Sources Unless Relevant

## Verbatim

> no need for new source untill we see enough relevance for one or its related to something we have or need or if I say so. otherwise lets continue in order.

## Interpretation

Don't proactively ingest new external sources. Only ingest when:
1. A source is directly relevant to an identified gap or open question
2. It's related to something the wiki already covers and would deepen it
3. The user explicitly requests it

Otherwise, work with what's already in the wiki — evolve, deepen, cross-reference, fill gaps from existing knowledge. Continue processing in order.
